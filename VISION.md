# TikTokBot Vision

`tiktokbot` should help George inspect TikTok breakout patterns without
pretending the platform exposes broad, stable public discovery to ordinary
operators.

## Product Thesis

The repo is useful when it makes the access reality explicit:

1. owned-account analytics are possible today through OAuth,
2. broad public discovery is constrained,
3. a manual worksheet scorer still preserves the core ranking logic.

## Goals

- Keep own-account outlier analysis reliable through the official path.
- Maintain a useful manual scoring workflow for public competitor research.
- Expose experimental web probing as a separate, brittle adapter.

## Non-Goals

- promising broad official public discovery when TikTok does not allow it
- collapsing manual, official, and unofficial paths into one misleading story
- turning the repo into a generic growth-hacking suite
